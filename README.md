# Spam Email Detection using Machine Learning

## Project Overview
This project develops a Spam Email Detection System using Machine Learning techniques. The system classifies emails as Spam or Not Spam based on their content.

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- TF-IDF Vectorization
- Logistic Regression Model
- Confusion Matrix Evaluation
- Classification Report
- Real Email Prediction Testing

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Model Performance
- Accuracy: 95.7%
- Precision (Spam): 92%
- Recall (Spam): 95%
- F1-Score (Spam): 94%

## Sample Prediction

Input:
Congratulations! You have won $1000. Click here to claim your prize now!

Output:
SPAM

## Conclusion
The model successfully classifies spam and legitimate emails with high accuracy and demonstrates the practical application of Machine Learning and Natural Language Processing techniques.
